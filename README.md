# web-verify — IT/AI 职业情报分析

让 AI 在回答职业、市场、技术趋势类问题之前，先搜一手信源、过滤营销噪音、标注可信度——再给你结论。

专为中国 IT/AI 就业市场设计。5 次独立冷启动测试通过。

## 什么时候用

向 AI 问这些问题时：
- "学 LangChain 能找到工作吗？"
- "北京 AI 应用开发薪资多少？"
- "听说 AI 岗位涨了 312%，真的假的？"
- "Java 还能不能学？"
- "哪个开源项目值得投入时间？"

普通 AI 凭训练数据给观点。加载 web-verify 后，AI 会先搜一手信源再回答。

## 安装

```bash
cp -r web-verify ~/.trae/skills/
```

## 使用

```
Use Skill: web-verify
```

然后直接问你的问题。AI 会自动执行：搜索 → 信源过滤 → 对抗验证 → 标注可信度 → 区分事实与推断。

## 调用链

作为基础层 skill，被其他 skill 调用用于市场数据搜验：

```
web-verify（基础层：搜索 + 信源过滤 + 对抗验证）
    ├──→ career-advisor（搜索市场数据）
    └──→ study-planner（搜索最新技术信息）
```

## 已验证

5 次独立冷启动测试全部通过：对抗验证（拆穿假数据）、Java 转 AI 咨询、LangChain 学不学（含 [DISPUTED]）、北京 AI 薪资、阿里/腾讯财报 AI 收入分析。
